# Fetch API Simple GET Request

This example demonstrates how to use the JavaScript Fetch API to make a simple GET request to a public API. It fetches a sample ToDo item from JSONPlaceholder and displays the received JSON data on the web page. The code illustrates the Promise-based nature of Fetch, including handling successful responses and basic error handling.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in any modern web browser.
